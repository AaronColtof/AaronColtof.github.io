Selected Projects

Siege of the Void

A Skylanders-inspired quest-based game built in a team, centered around real-world NFC interaction. As part of a self-directed research project, we explored the technology behind Skylanders and recreated the full experience, from custom NFC figures and a working portal to a complete playable game. I contributed to both the software and system design, combining hardware and gameplay into one cohesive experience.

Unity C# NFC Arduino

Multiplayer Space Shooter

A real-time 3D multiplayer space shooter developed in a team using Unity and FishNet networking. Players can connect from anywhere and compete to destroy each other’s base. I worked on core gameplay systems and multiplayer functionality, focusing on synchronization and smooth online interaction.

Unity C# FishNet Multiplayer networking

Maze Generator

A procedural maze generator written in C++ as part of a university-level course. The program generates a new random maze each run, guaranteeing that it is always solvable. This project focused on algorithms, data structures, and efficient memory usage.

C++ Algorithms Procedural Generation
